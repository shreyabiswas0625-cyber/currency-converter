# CoinConvert 💱

A beginner-friendly Currency Converter web application built using HTML, CSS, and JavaScript. The application allows users to convert between different currencies using real-time exchange rates fetched from an API.

## Features

- 🌍 Convert between multiple international currencies
-  Fetches live exchange rates
-  Automatically updates country flags based on selected currencies
-  User-friendly interface with responsive design
-  Fast and lightweight

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API
- Flags API

## Preview

Enter an amount, select the source and target currencies, and click **Get Exchange Rate** to view the converted value instantly.
